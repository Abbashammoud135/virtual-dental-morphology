# 🦷 Virtual Dental Morphology

**An immersive Virtual Reality application developed to transform traditional dental morphology practical sessions into a fully interactive 3D experience.**

This project is a collaboration between the **Faculty of Engineering** and the **Faculty of Dentistry** at the **Lebanese University**.

---

## 🎯 Project Overview

Traditional dental morphology labs require students to manually construct teeth models during practical sessions.  
This project converts the entire practical experience into **Virtual Reality** using **Meta Quest 3**.

Students can:

- **Construct teeth from scratch** in a virtual environment  
- **Interact with 3D dental structures** in real time  
- **Improve spatial understanding** of tooth anatomy  
- **Practice morphology** without physical material limitations  

---

## 🧠 Educational Impact

The platform enhances:

- **Hands-on learning** in a safe virtual space  
- **3D visualization** and spatial perception  
- **Repetitive practice** without material waste  
- **Immersive and engaging** learning experience  

---

## 🛠 Technologies Used

- **Virtual Reality (VR)**

- **3D modeling and interactive simulation**  
- **Game engine integration** (Unity)
